cutCircle(center, radius, depth, cutProperties, safeZopt) → {string|boolean}

Generates G-Code for cutting a circle. The circle is considered 2D on the XY plane. Do not use this function if you cut completely through the material: you need tabs for that. Without tabs, the cutting part can be thrown because of the spindle rotation.
Parameters:
Name Type Attributes Description
center Vector The circle center.
radius number The radius in inches.
depth number The depth in inches.
cutProperties CutProperties The cut properties.
safeZ number <optional>
 The safe Z position to go after the cut in inches.

cutPolygon(polygon, depth, cutProperties, safeZopt) → {string|boolean}

Generates G-Code for cutting the polygon. The order of the polygon tips is important. The bit will go to a point to the next one and close the polygon by going from the last point to the first point. The polygon is considered 2D on the XY plane. Do not use this function if you cut completely through the material: you need tabs for that. Without tabs, the cutting part can be thrown because of the spindle rotation.

pocketConvexPolygon(polygon, depth, cutProperties, safeZopt) → {string|boolean}

Generates G-Code for pocketing the convex polygon. The order of the polygon tips is important. The bit will go to a point to the next one and close the polygon by going from the last point to the first point. The polygon is considered 2D on the XY plane. If the polygon is not convex, the behaviour is undefined.
